Privacy Policy

Last updated: July 2, 2026

UKAF App ("the app") is a desktop application used internally by clinic staff to manage patient visits and insurance authorization records. This policy explains what data the app handles and, specifically, how it uses Google Drive access.

Data stored by the app

All clinic operational data — patient visit records, insurance status, stamped documents, audit logs — is stored locally on the clinic's own computer, in a local database file. This data is never transmitted to any server operated by the app's developer. The developer has no access to this data.

Google Drive access

The app offers an optional "Google Drive backup" feature. If a clinic administrator chooses to connect their Google account, the app requests only the https://www.googleapis.com/auth/drive.file scope. This scope is intentionally narrow:

  • The app can only see and manage files that it itself creates in the connected Google Drive account (backup .zip files, stored in a folder named "UKAF App Backups").
  • The app cannot browse, read, or modify any other file already in the user's Drive.
  • No data from the backup files is ever sent to or read by the app's developer, or any third party. Only the connected Google account itself can access the files, in its own Drive.

Authentication data

OAuth tokens obtained during the Google sign-in process are stored encrypted, locally on the clinic's computer (using the operating system's built-in credential encryption), and are used solely to authenticate the app's own backup uploads to that Drive account. Disconnecting the Google account in the app deletes these tokens immediately.

Data the developer collects

The developer of UKAF App does not operate any server that receives clinic data, patient data, or Google account data. No analytics or tracking is embedded in the app.

Changes to this policy

If this policy changes, the updated version will be published at this same URL with a new "Last updated" date.

Contact

Questions about this policy or the app's data handling can be sent to ukafapp@gmail.com.
