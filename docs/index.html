UKAF App

Universal Clinical Assessment Form — a desktop app for managing clinic visits and insurance authorization workflows.

UKAF App is a Windows desktop application used internally by clinic staff to manage patient visits, insurance authorization workflows, doctor stamping of forms, and related administrative records. It runs locally on each clinic's own computer and keeps its data in a local database on that machine — nothing is sent to a server operated by the developer.

What it does

  • Tracks patient visits, insurance authorization status, and related documents
  • Lets doctors apply digital stamps to insurance forms (UCAF / PDF documents)
  • Tracks KPIs around insurance claim approvals and rejections
  • Backs up its local data, optionally to the clinic's own Google Drive

Google Drive access

UKAF App can optionally back up its data as a single .zip file to the clinic's own Google Drive account. It uses Google's narrowest available Drive scope, drive.file, which only lets the app see and manage files that it itself creates — it cannot see, list, or modify anything else already in the user's Drive.
